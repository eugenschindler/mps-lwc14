This is the code for the MPS entry to the Language Workbench Challenge (LWC) 2014.
The initial code is the code from the LWC 2013 entry for MPS.

To properly run the languages and models, please install the following 
plugins (File -> Settings -> Plugins -> Browse Repositories):
  * MPS Editor Override
  * MPS Multiline Property Editor
  * MPS Richtext Editor Component
